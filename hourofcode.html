Hour of Code Event

On November 7th, our team visited School #229 to teach fifth-grade students the basics of coding using code.org as part of the Hour of Code project. The session included hands-on activities from the Angry Birds course, with all students actively participating. After completing the tasks, we awarded certificates to recognize their achievements in learning coding fundamentals. The event aimed to make coding fun and accessible while sparking interest in the students.
